# Skill: Crear HTML5

## Objetivo
Generar estructura HTML5 semántica, accesible y bien organizada.

## Instrucciones
- Siempre comenzar con <!DOCTYPE html> y el atributo lang="es".
- Incluir en <head>: charset UTF-8, viewport para responsive, meta description y title.
- Usar etiquetas semánticas: <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>.
- Vincular la hoja de estilos con: <link rel="stylesheet" href="styles/main.css">
- Vincular el script con: <script src="scripts/main.js" defer></script>
- Usar atributos id y class con nombres descriptivos en inglés o español consistente.
- Incluir atributo alt descriptivo en todas las imágenes.
- Estructurar formularios con <label>, <fieldset> y <legend> cuando corresponda.
- No incluir estilos inline ni scripts inline; todo va en archivos separados.

## Formato de respuesta
Entregar siempre:
1. Archivo index.html completo con DOCTYPE y estructura semántica.
2. Referencias correctas al CSS en styles/main.css y al JS en scripts/main.js.
3. Contenido representativo (no dejar secciones vacías o con Lorem Ipsum vacío).
