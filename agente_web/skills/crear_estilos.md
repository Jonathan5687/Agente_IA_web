# Skill: Crear hojas de estilo CSS

## Objetivo
Generar CSS limpio, organizado, mantenible y responsive.

## Instrucciones
- Definir variables globales en :root para colores, tipografías y espaciados.
- Incluir un reset básico al inicio (* { margin: 0; padding: 0; box-sizing: border-box; }).
- Usar Flexbox o CSS Grid para los layouts principales.
- Organizar el archivo en secciones claramente comentadas:
  1. Variables (:root)
  2. Reset / Base
  3. Layout general
  4. Header y Navegación
  5. Contenido principal
  6. Componentes (botones, tarjetas, formularios, etc.)
  7. Footer
  8. Media queries (al final)
- Usar nombres de clases descriptivos y consistentes.
- Definir estilos para estados interactivos: :hover, :focus, :active.
- Evitar el uso de !important salvo casos excepcionales justificados.
- Implementar al menos un breakpoint responsive (mobile: max-width 768px).

## Formato de respuesta
Entregar:
1. Archivo styles/main.css completo con todas las secciones comentadas.
2. Variables definidas en :root.
3. Diseño responsive con media queries al final del archivo.
